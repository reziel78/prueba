Instalar el plugin

1. Descargar archivo ZIP del plugin

Descargamos el archivo de la página del autor. Descomprimimos el archivo y veremos una carpeta con un nombre bastante largo. Le cambiamos el nombre de la carpeta principal a "timepicker".

2. Código para poner en el HEAD de nuestra página

Ahora debemos insertar en la cabecera (HEAD) de nuestra página web varios enlaces:

  1. El enlace de la librería principal JQUERY.
  2. Enlace al archivo Javascript del plugin.
  3. Enlace al archivo CSS del plugin (en algunos plugin no hace falta).

De todas formas, hay que leer la documentación del Plugin para que nos indique qué archivos hay que poner en el HEAD, porque cada Plugin es diferente.

3. Insertar los objetos HTML

Debemos consultar la documentación del autor para saber cómo activar los eventos del plugin. Normalmente, se le asigna un "id" a la etiqueta HTML donde ocurrirá el cambio y, en el mismo documento HTML, se le añade un < script > que ejecuta la acción. En este ejemplo, la id #setTimeExample es la que identifica al elemento INPUT que activa el desplegable con las horas.
